# friendslop-ezpz

A small game project made with Godot using the easy-peasy plugin.

## About

**friendslop-ezpz** is a lightweight Godot game built with GDScript.  
It focuses on simple, fun gameplay and serves as a compact project for experimenting with the Godot easy-peasy plugin.

## Features

- Built with Godot
- Written in GDScript
- Uses the easy-peasy plugin
- Simple and lightweight game project

## Getting Started

1. Open the project in Godot.
2. Make sure the easy-peasy plugin is installed and enabled.
3. Run the project from the Godot editor.
