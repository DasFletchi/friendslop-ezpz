# TL;DR

Die Guard-Silhouette muss:
- sofort lesbar sein (1 Blick = Gefahr erkannt)
- klare Form-Archtypen haben (Brick / Pole / Shield / Signal)
- Helm als Identität nutzen (kein Gesicht nötig)
- starke negative spaces haben (kein Blob-Matsch)
- ein ikonisches funktionales Element besitzen (Licht, Schild, Antenne etc.)
- aus der Distanz erkennbar bleiben
